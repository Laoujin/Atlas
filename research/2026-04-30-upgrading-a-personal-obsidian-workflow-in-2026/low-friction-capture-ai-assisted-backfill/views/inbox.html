Desktop quick capture

The 2026 baseline is unchanged from 2024: today's daily note is the inbox, processing happens at review time, capture is one hotkey away[12]. What changed is that the two plugins doing the work are both still actively maintained and explicitly complementary — QuickAdd handles menu/prompt/routing, Templater handles dynamic variables and JS[7].

  • QuickAdd ⭐ 2.2k v2.12.0 (Mar 5 2026) — Capture, Template, Macro, Multi choices[1][2]
  • Templater ⭐ 4.9k v2.20.0 (Apr 29 2026) — JS execution + per-template hotkeys[3][4]
  • Journals (srg-kostyrko) — actively-maintained 2025/2026 Periodic Notes replacement; daily→yearly + interval journals + bulk import[6]
  • Periodic Notes ⭐ 1.3k — last release Apr 2022; treated as effectively unmaintained[5]
  • Thino ⭐ 1.3k — closed-source from v2.0.0 with paid pro[11]
System-wide hotkey. Obsidian's own hotkeys only fire when Obsidian has focus. Cross-platform: QuickAdd + the Global Hotkeys plugin, Ctrl+Alt+A is a common binding[8]. Windows: AutoHotkey script bound to Ctrl+Alt+V activates Obsidian and forwards Ctrl+Alt+C into QuickAdd's capture window[9]. macOS: Alfred/Raycast workflows hitting QuickAdd's URI[8].
Inbox-processing pattern. Advanced URI + Templater + Hotkeys for Templates with Opt+A/Opt+D/Opt+M bindings to archive, trash, or move each captured fleeting note during a daily review pass[10]. Capture is dumb and fast; processing is concentrated.

Mobile capture

The headline 2026 shift: Obsidian Mobile finally shipped first-party capture surfaces — but architectural friction remains. Even widget taps still cold-start the full vault, so iOS power users still front-end with Drafts or Actions for Obsidian.

  • Mobile 1.11 (Jan 2026) — iOS Lock Screen / Control Center / Home Screen widgets; Android home-screen widgets + Quick Settings Tile; Siri "Capture using Obsidian"; "Capture to a note" Shortcut[14]
  • Mobile 1.12 (Feb 27 2026) — native iOS Share extension; "Bookmark Link" Shortcut; "Unique note" home-screen long-press[13]
  • Drafts (since v28 Bookmarks) — writes Markdown into iCloud-stored vault without launching Obsidian; Watch complication[19]
  • Actions for Obsidian (iOS 18.4+) — 40+ Shortcuts actions: append, prepend, create-from-template, fetch frontmatter[17]
  • Quick Draft — free Android/iOS companion: text/voice/image, OCR, date-formatted paths, template wrapping, Siri shortcuts[18]
  • Tasker (Android) — direct-write timestamped Markdown into vault folder; sync only fires when Obsidian opens[20]
  • Fleeting Notes — Android-first scratch-note frontend that syncs into the vault[24]
  • Apple Watch — still no first-party Obsidian app in 2026; Drafts complication / Siri Shortcuts via iCloud / third-party Uplink[23]
2026 Report Card scores mobile 3.1/5 — widget taps still cold-start the full vault; community sentiment "still awkward and kludgy" despite 1.11/1.12[16]. The two-tier flow holds: short thoughts via iOS Shortcut popup + #QuickNotes append; longer captures via Drafts[22].

Web, article, email, RSS

Omnivore's November 2024 shutdown after the ElevenLabs acquihire reset the field[29]. The official Web Clipper with the Interpreter LLM template-fill replaced Omnivore, MarkDownload, and most third-party clippers.

  • Obsidian Web Clipper ⭐ 4.1k v1.6.2 — official cross-browser (Chrome/Safari/Firefox/Edge/Brave/Arc/Orion/Vivaldi + mobile); templates for articles/recipes/references/academic[25][26]
  • Interpreter — pipes page context + {{"prompt"}} to BYO-LLM (Claude/GPT/Gemini/local) for summary/tags/author fields[27]
  • Readwise ⭐ 335 — highlight-centric: Kindle/Apple Books/Pocket/Twitter/PDFs/Reader; V2 added mobile sync + MCP server[30][31]
  • MarkDownload ⭐ 3.8k — generic page→Markdown via Advanced Obsidian URI fallback when Web Clipper templates don't fit[32]
  • Hypothes.is annotations — weichenw ⭐ 270 (one-way) or lindylearn fork (bi-directional, vault edits propagate back)[35][36]
  • Email → vault — Email2Obsidian (free 10 lifetime, $2.49/mo Starter) or TaskRobin (forwarded emails + attachments)[37][38]
  • RSS Dashboard ⭐ 499 — aggregates RSS + YouTube + podcasts with smart tagging and embedded media[40]
  • joethei RSS Reader ⭐ 468 — archived August 25 2025; last release v1.2.2 (June 2022)[39]
  • Matter ⭐ 128 — Obsidian plugin inactive since November 2022; Matter Premium $60/yr is still maintained but the plugin isn't[33][34]
Self-hosted escape hatch. Wallabag for users who want full data ownership rather than another SaaS[41]. Raindrop.io works as a triage inbox alongside Web Clipper but does not store full Markdown bodies[28].

Voice-to-note

A clear hierarchy emerged in 2026. Whisper is the engine; nikdanilov's plugin is the in-vault default. iOS 18+ Voice Memos on-device transcription handles short captures well enough that many users skip Whisper for them.

  • nikdanilov/whisper-obsidian-plugin ⭐ 350 v1.9.1 (Apr 2026) — OpenAI/Groq/Azure/Ollama/LM Studio; hotkey recording (Alt+Q); LLM post-processing; URL automation for iOS Shortcuts[42]
  • iOS 18+ Voice Memos — on-device transcription on iPhone 12+, ~85–90% accuracy on clean single-speaker[49]
  • Vox ⭐ 78 — folder-watcher with filename-prefix metadata tokens; dev's free server (100/day); no API keys needed[51]
  • Smart Memos ⭐ 79 — OpenAI-only one-shot transcript+summary+expansion; local-model on roadmap[44]
  • Wispr Flow — system-wide voice keyboard; vendor claims ~220 wpm vs 45 wpm typing (~4×)[52]
  • AudioPen → Obsidian ⭐ 7 — webhooks with templated {title}/{body}/{orig_transcript} for users already paying for AudioPen[50]
  • djmango/obsidian-transcription ⭐ 222 — last release v3.4.0 (Apr 2024); stagnant[43]
Cost & accuracy. OpenAI Whisper API ~$1 per 2h45m of audio with strong accent handling[45]. Local whisper.cpp: reduced accuracy, occasional accent failures, 16GB RAM floor[45]. Whisper Large-v3: 2.7% WER on clean English / 8–12% on real-world / 8–15% on accented[46].
Forum pipeline. The 2026 r/ObsidianMD pattern is a four-stage pipeline: voice memo → transcription → AI summary → structured note → vault. Friction concentrates around fragmented recordings (interruptions, Bluetooth switching) rather than transcription quality[47]. iPhone DIY: iOS Shortcuts + RoutineHub piping Voice Memos through Whisper (cloud or local Aiko) and ChatGPT, then iCloud → vault[48].

AI-assisted metadata backfill

The 2026 field has fragmented into lanes — there is no single winner. Pick the lane that matches the job. AI Tagger Universe is the clearest fit when the job is batch tag/frontmatter generation over an existing vault.

  • AI Tagger Universe ⭐ 84 v1.0.17 (Mar 2026) — batch tag entire folders or whole vault; nested hierarchical tags into frontmatter; 15+ providers (Ollama, LM Studio, OpenAI, Claude, Gemini, DeepSeek, Bedrock, Vertex AI)[53]
  • Note Companion ⭐ 832 v3.6.18 — folder/tag/title/template suggestions; inbox-automation processes raw captures in batch; cloud sub, self-host, BYO key; desktop-only[59][63]
  • Text Generator ⭐ 1.9k — template-driven generation; reusable prompts + frontmatter-configurable settings to batch-generate metadata/summaries/aliases[58]
  • Metadata Auto Classifier ⭐ 52 — 2025 entrant; OpenAI GPT only; classifies tags + custom frontmatter fields by user rules[60]
  • LLM Tagger — Ollama-only auto-tagging on new/modified notes; lighter alternative for fully-local users[64]
  • Frontmatter Generator ⭐ 46 — non-AI deterministic JS/JSON template fallback; whole-vault and per-folder commands[65]
  • Smart Connections ⭐ 4.9k — semantic discovery via embeddings, not backfill; source-available license + 2026 pricing complaints[54][55]
  • Copilot for Obsidian ⭐ 6.8k — agentic chat + RAG + Composer V2; not a dedicated backfill tool[56][57]
Hands-on validation. A 2026 MakeUseOf walkthrough paired AI Tagger Universe with Auto Note Mover and locally-hosted Gemma 3 12B (LM Studio) — the Generate tags for vault command retro-tagged hundreds of existing notes in under an hour of setup[61].
Reddit splits picks across three lanes. Smart Composer (fastest, free Llama 3.2) for writing assist; Smart Connections for semantic search; Copilot for vault-wide RAG chat[62]. New entrants on the agent-and-retrieval frontier: Obsilo Agent (55+ tools), Sonar (local-first)[63].

Local-first AI backbone

The 2026 case is concrete: notes never leave the machine, the workflow runs offline, and there is no per-token cost once the model is downloaded[80]. Two engines and two GUIs cover essentially the field.

  • llama.cpp ⭐ 100k — crossed 100K stars March 2026, faster than PyTorch (~7y) or TensorFlow (~8y)[71]
  • Ollama ⭐ 110k — wraps llama.cpp's inference backend; on Apple Silicon now built on MLX[72][74]
  • LM Studio (closed-source) — built-in HuggingFace browser; OpenAI-compatible server on localhost:1234/v1[79]
  • whisper.cpp ⭐ 49k — C/C++ port of Whisper; offline GPU-accelerated[75]
  • Apple Foundation Models SDK — Python bindings ⭐ 1.0k (apple/python-apple-fm-sdk); Swift API exposes guided generation, constrained tool calling, LoRA adapters[77][78]
Apple Silicon throughput (Llama 3.1 8B). M1/M2/M3/M4 base hit 15–28 tok/s through Metal; M3 Pro/Max with 18+ GPU cores reach 28–35 tok/s via unified memory[73]. ⚠ Single vendor-blog source — corroborate against Ollama's own benchmarks before sizing hardware.
Recommended models for tagging/summary 2026: Llama 3.3 70B (40GB VRAM, 86.0 MMLU) top quality; Qwen 2.5 32B (20GB, 83.2 MMLU) best mid-range; Llama 3.1 8B (6GB) default for 8GB box; Mistral 7B v0.3 (6GB) terse summary specialist for emails/meetings[69][70].
Plugin integration. Both dominant AI plugins speak local. Copilot needs OLLAMA_ORIGINS=app://obsidian.md* exported to bypass CORS; pull nomic-embed-text or mxbai-embed-large for fully offline RAG[66]. Smart Connections embeds locally by default — 786K community plugin downloads (Jan 2026)[67][68]. Apple Intelligence Writing Tools partially exposed in Obsidian's Edit menu on macOS 15.1+[76]; no plugin yet wraps the Foundation Models SDK directly — open opportunity for 2026.
