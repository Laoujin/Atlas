Operational Warnings 4 GOTCHAS · READ BEFORE INSTALLING

⊠ HARD RULE

Don't double-index

Running Smart Connections + Copilot Vault QA + Smart Composer RAG simultaneously means three separate embedding indexes over the same vault[13]. Pick one chat plugin; let Smart Connections handle the discovery sidebar.

⏱ FIRST-RUN COST

20–30 min on a 5,000-note vault

First-build embedding indexing locally takes 20–30 minutes on a 5k-note vault. Index lives in .obsidian/ and updates incrementally afterward[15]. Plan for the first run, not the steady state.

▣ KNOWN-GOOD STACK

Local-only: Ollama + nomic-embed-text + Qwen 2.5

2026's known-good local stack: Ollama + nomic-embed-text for embeddings + Qwen 2.5 (or Llama 3) for chat, wired into Copilot or Smart Composer[6][14].

⚠ IMMATURE LANE

Agentic plugins are early

Plugins like Obsidian Chat and Vault AI Chat can read+edit+create files autonomously[20] — interesting but immature. The more reliable 2026 pattern: wire Smart Connections retrieval into Claude Code or another external agent via MCP[19].

Install Order THE TWO-PLUGIN BASELINE + OPTIONAL ADDS

01
Install Smart Connections

Local-first sidebar. Zero config. Index first. Offline after.

02
Pick ONE chat plugin

Copilot for polish · Smart Composer for editing · Khoj if cross-device.

03
Stand up Ollama

Pull nomic-embed-text + Qwen 2.5 / Llama 3. Point chat at localhost:11434.

04
Resist double-indexing

Disable any second RAG index. Smart Connections owns discovery.

05
(Optional) Wire MCP

Expose retrieval to Claude Code or external agents. The 2026 advanced pattern.
